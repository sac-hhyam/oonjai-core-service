# Feature 2 — Caretaker Selection / Search

**Feature under test:** Caretaker browse, filter, sort and selection flow (reached after the user submits a service request).

**Goal:** Verify that the user can browse, filter, and select a caretaker from the list, and that edge cases such as empty results or unavailable caretakers are handled gracefully.
