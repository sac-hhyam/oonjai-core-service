# Feature 3 — Booking Creation and Confirmation

**Feature under test:** Booking summary review, submission, and confirmation page.

**Out of scope:** Payment, checkout-money, refunds, and card flows are NOT implemented yet and are explicitly excluded from these tests.

**Goal:** Verify that the user can review the booking summary, submit a booking, and reach the confirmation page, while invalid attempts are rejected without creating a booking.
